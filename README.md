# Velhari Sankaran — Founder Journey

A mobile-first public dashboard documenting Velhari Sankaran's education, financial stability, business journey, and fundraising milestones.

## Source of truth

The first page reproduces the Google Stitch **Velhari Home Dashboard** from project `1319779829649559419`, screen `1e656a82a14343fa84c6dd7beb52d43a`.

## Local development

```bash
npm install
npm run dev
```

Run `npm run build` before publishing. Pushes to `main` deploy automatically through GitHub Pages.
