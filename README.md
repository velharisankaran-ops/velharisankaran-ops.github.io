# Velhari Sankaran — Founder Journey

A mobile-first public dashboard documenting Velhari Sankaran's education, financial stability, business journey, and fundraising milestones.

## Source of truth

The site reproduces two mobile screens from Google Stitch project `1319779829649559419`:

- **Velhari Home Dashboard** — screen `1e656a82a14343fa84c6dd7beb52d43a`
- **Plus Two Goal Details with Contributor Ledger** — screen `9acd80ff99e641f48dbde556c2a683b8`

The goal screen is available at `#/Fundraising-goals/Acadamics` and is linked from the Home fundraising card.

## Local development

```bash
npm install
npm run dev
```

Run `npm run build` before publishing. Pushes to `main` deploy automatically through GitHub Pages.
